# 総合テスト仕様書

**システム名：** LumberCore 木材受発注基幹システム
**対象範囲：** フロントエンド（Vercel） + バックエンド（Spring Boot）を組み合わせたシステム全体
**テスト方式：** ブラウザ手動確認（Chrome/Edge）
**作成日：** 2024-04

> **総合テストの方針**
> ユーザー視点のエンドツーエンドシナリオで、業務フロー全体が正しく動作することを確認する。
> 個別の単体・結合テストでは検出しにくい「画面遷移」「データの一貫性」「異常データ耐性」「UI/UX品質」を重点的に検証する。
>
> **テスト環境前提条件**
> - フロントエンド：Vercel 本番URL または `localhost:5500`
> - バックエンド：Spring Boot `localhost:8080` または本番ホスティング先
> - ブラウザ：Chrome 最新版（開発者ツールを使用）
> - ダミーデータが初期表示されること
